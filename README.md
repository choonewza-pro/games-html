# Games HTML

รวมเกม HTML แบบโต้ตอบด้วยมือผ่านกล้อง (AR Hand Tracking)

## เกมที่มี

- **Balloon Burst v1** — เกมเป่าลูกโป่ง แบบคำประวิสรรชนีย์ และคำไม่ประวิสรรชนีย์
- **Balloon Burst v2** — เกมเป่าลูกโป่งออนไลน์แบบผู้เล่นหลายคน (Multiplayer)

## เทคโนโลยี

- MediaPipe Hands (Hand Tracking)
- Tailwind CSS
- Firebase Realtime Database (v2)
